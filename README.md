## Installation

Download the repo:

```
git clone https://github.com/BetaHuhn/portfolio
```

Change directory:

```
cd portfolio
```

Install Vue.js dependencies:

```
npm install
```

Build frontend:

```
npm run build
```

Development

```
npm run serve
```
